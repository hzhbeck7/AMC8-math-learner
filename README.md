# AMC8-math-learner
using AI to help your kid learn AMC8 math knowledge AI化身奥数教练帮助你的孩子学习AMC8

上传PDF或JPG，AI会自动帮你把题目分析并给出解答思路
Upload any PDF or JPG that contains questions. AI will help your kid analyze the question step by step and give answer, also it will show relavant knowledge and interesting math history/story that related with this question.
